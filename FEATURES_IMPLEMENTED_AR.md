# ملخص المزايا الأساسية (محدث للنسخة v142)
- واجهة رئيسية فاخرة مستوحاة من التصميم المعتمد، دون نسخ علامة أو أصول جواكر.
- شريط حساب: مستوى، عملات، أيام الباشا.
- بطولة رئيسية، طريق هدايا، ألعاب مميزة، مسابقات ومباراة ودية.
- متجر تفاعلي: باشا، ثيمات، ظهر بطاقات، إيموجي، مسرعات، ألوان أسماء.
- غرفة طرنيب تجريبية: طاولة، أوراق، لاعبين، نقاط، مؤقت، طلب/سكون/لعب.
- دردشة غرفة وتفاعلات سريعة مخفية بزر وتفتح/تغلق.
- إشعارات داخل قائمة منبثقة.
- ست لغات مع RTL/LTR، وخمسة ثيمات محفوظة محلياً.
- صفحات ألعاب، أندية، مسابقات، متجر.
- API client وLaravel Sanctum endpoints كبداية للربط الحقيقي.
- GitHub Actions للويب وAndroid وiOS.
