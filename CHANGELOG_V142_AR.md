# سجل تغييرات Warqna v142

## محركات الألعاب

- دمج محرك الطرنيب المستقل المرفق.
- دمج محركات طرنيب سوري و400، تركس، هاند، بناكيل، بلوت وسوليتير من الحزمة المرفقة.
- إنشاء Registry موحد للألعاب وقواعدها وحركاتها.
- تطوير محركات دومينو وباصرة وجاكارو وطاولة الزهر وشطرنج.
- إضافة توزيع آمن Fisher–Yates باستخدام `random_int`.
- إضافة اختبارات مستقلة للمحركات والقواعد الأساسية.
- تحسين البوت عند انتهاء الوقت ليختار حركة قانونية.
- إظهار أسماء البوتات واللاعبين في سجل اللعب.

## Flutter

- شاشة دخول/تسجيل/خروج وربط Sanctum.
- وضع محلي تجريبي للمدير.
- غرف لعب خادم متجاوبة للطول والعرض.
- تحسين قائمة الحركات القانونية لجاكارو والطاولة والدومينو والشطرنج.
- إظهار اسم صاحب الورقة في اللمّة.
- متجر مع تأكيد شراء ومعاينات وفئات.
- مركز أصدقاء ودردشة خاصة وطلبات وحظر وتحويل توكنز.
- لوحة إدارة داخل التطبيق.
- ثيمات ولغات وحفظ الخيارات.

## Backend

- Mobile API للحسابات والملف والمحفظة والمتجر والإشعارات والمكافآت.
- API لإنشاء جلسات الألعاب والحركات والمؤقت والدردشة والمغادرة.
- API للأصدقاء والدردشة والتحويل بعمولة 10%.
- API إداري للألعاب والمتجر والمستخدمين.
- حماية أوراق الخصوم من استجابة العميل.
- تسجيل كل حركة وقبولها/رفضها.
- اللعب مجاني، ورسوم دخول Mobile API تساوي صفراً.
- حساب Adnan برصيد `1000000000000000000` توكن.
- Migration لمنتجات المتجر وأسعارها ومعايناتها.
- Migration لمزامنة كتالوج المحركات مع لوحة الإدارة.
