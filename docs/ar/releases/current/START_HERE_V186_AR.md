# ابدأ من هنا — Warqnaa V0.3.5 / v186

الإصدار التقني: **0.3.5+186**.

هذا الإصدار يبني فوق V185 ويحافظ على المتجر والطاولات والاقتصاد والنوادي والصوت والمكافآت، ويخصص التحديث لتوحيد وفحص محركات الألعاب الثماني عشرة الظاهرة في Flutter.

## أهم التغييرات

- كتالوج واحد ثابت من 18 لعبة بين Flutter وLaravel وسجل المحركات.
- رفض أي مفتاح لعبة مجهول بدل تشغيل محرك عام متساهل.
- تحقق كامل من بيانات الحركة داخل المحرك قبل حفظها.
- قفل معاملات اللعب وانتهاء الوقت لمنع احتساب حركتين متزامنتين.
- إصلاح هاند 2–5 لاعبين، دورة 15/14، السحب والتنزيل والرمي والنقاط.
- إعادة بناء سوليتير التنافسي برزمة Klondike مستقلة وكاملة لكل لاعب.
- تصحيح مراحل شراء البلوت والأهداف، وتقوية إجبار النرد في طاولة الزهر.
- اختبارات مستقلة تغطي تهيئة كل محرك وحركته القانونية ورفض الحركة خارج الدور.

## التشغيل والفحص

1. على Windows شغّل `CHECK_WARQNA_WINDOWS.bat`.
2. بعد نجاح الفحص شغّل `START_WARQNA_WINDOWS.bat`.
3. على Linux/macOS شغّل `bash scripts/unix/current/check-v186.sh`.
4. لا تنشر قبل نجاح GitHub Actions للويب وAndroid وiOS وLaravel.

## حدود التحقق المحلي

فحوص المصدر وAST والعقود نجحت محليًا. تشغيل `flutter analyze` و`flutter test` و`php artisan test` واختبارات PHP التنفيذية الكاملة يحتاج Flutter وPHP/Composer، ولذلك يبقى GitHub Actions بوابة النشر النهائية.
