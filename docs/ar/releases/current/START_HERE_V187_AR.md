# ابدأ من هنا — Warqnaa V0.3.6 / v187

الإصدار التقني: **0.3.6+187**.

هذا الإصدار يبني فوق V186 ويحافظ على المتجر والطاولات والاقتصاد والنوادي والصوت والمكافآت، ويضيف طبقة لعب محلي كاملة للألعاب الظاهرة مع تدقيق CI جديد.

## أهم التغييرات

- توحيد كتالوج Flutter وLaravel وسجل المحركات على 20 لعبة.
- تشغيل جميع ألعاب الكتالوج محليًا مع بوتات عند غياب الخادم.
- إضافة جاكارو وليخة بمحركات مستقلة وقواعد حركات واختبارات انتقالات.
- إعادة بناء دومينو وطاولة الزهر وسوليتير التنافسي محليًا بدل الاعتماد على مسار خادم.
- دعم ترتيب ورق اللاعب بالسحب والإفلات.
- دعم اختيار مجموعات هاند وبناكل يدويًا، تنزيل عدة مجموعات، والتركيب على مجموعة موجودة.
- إظهار كل مجموعة منزلّة داخل حاوية بلون مختلف مع مالكها وقيمتها.
- تبويب سلامة المحركات في لوحة الإدارة يعرض حالة Offline/Online ونسخة القواعد.
- إصلاح سياسة الجذر النظيف لتقبل حصريًا مخلفات حزم V142/V143 المعروفة.
- إصلاح اختبار V183 بحيث يختار صاحب الدور الحقيقي في عقد تركس.

## التشغيل والفحص

1. على Windows شغّل `CHECK_WARQNA_WINDOWS.bat`.
2. بعد نجاح الفحص شغّل `START_WARQNA_WINDOWS.bat`.
3. على Linux/macOS شغّل `bash scripts/unix/current/check-v187.sh`.
4. لا تنشر قبل نجاح GitHub Actions للويب وAndroid وiOS وLaravel.

## ملاحظة أمان

اللعب المحلي لا يحتاج خادمًا، لكنه مخصص للعب على الجهاز مع البوتات. الغرف التنافسية والاقتصاد والمشتريات والنتائج المصنفة تبقى خادمية وموثوقة، لأن اعتمادها على الهاتف وحده يتيح الغش.
