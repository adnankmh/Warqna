# ابدأ من هنا — Warqnaa V0.3.4 / v185

الإصدار التقني: **0.3.4+185**.

يضيف هذا الإصدار متجر طاولات متجاوبًا بهوية زمردية وذهبية، ويعرض مجموعة الطاولات الأربعين المعتمدة في التصميم الجديد ضمن مجموعات 1–10 و11–20 و21–30 و31–40، مع الحفاظ على جميع الطاولات القديمة ليبقى الإجمالي 140 طاولة.

الصورة المرجعية الأصلية مضمّنة داخل أصول Flutter، وتُعرض الطاولات منها بقصّ مضبوط عند الحاجة حتى لا تظهر المعاينات الموروثة الصغيرة أو الشفافة.

## التشغيل

1. شغّل `CHECK_WARQNA_WINDOWS.bat`.
2. بعد نجاح الفحص شغّل `START_WARQNA_WINDOWS.bat`.
3. للنشر الفعلي اضبط `WARQNA_API_URL` ومتغيرات Laravel وFirebase وAdMob.
4. لا تنشر قبل نجاح GitHub Actions للويب وAndroid وiOS والخادم.

## الأمان

- الشراء والتفعيل في نسخة الإنتاج يعتمدان من Laravel.
- عند تعذر الوصول إلى الخادم لا يتم خصم الرصيد ولا تغيير الملكية محليًا.
- التفعيل يثبت ملكية العنصر للمستخدم ويعطل العنصر السابق من الفئة نفسها داخل معاملة قاعدة بيانات.
