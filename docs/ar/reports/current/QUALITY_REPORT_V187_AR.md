# تقرير جودة Warqnaa V0.3.6 / Build 187

الإصدار: **0.3.6+187**

## نطاق الإصدار

يوحّد V187 عشرين لعبة بين Flutter وLaravel، ويجعل كل لعبة في كتالوج اللاعب قابلة للعب محليًا مع البوتات، مع بقاء الخادم المرجع الوحيد للغرف المصنفة والاقتصاد والمشتريات.

| العائلة | الألعاب | أبرز التحقق |
|---|---|---|
| طرنيب | طرنيب، 41، 61، 400، السوري | التوزيع، الطلب، الحكم، اتباع النوع، اللمات والهدف |
| تركس | تركس، شراكة، كمبلكس | صاحب المملكة، العقود، اتباع النوع، التدبيل والنقاط |
| هاند/بناكل | هاند، سعودي، شراكة، بناكل، كلاسيك | 15/14 أو 19/18، السحب، المجموعات، التركيب، الرمي والنقاط |
| خليجي/التقاط | بلوت، باصرة | شراء البلوت المرحلي، الصن/الحكم، الالتقاط وهدف الباصرة |
| فردية/لوحية | سوليتير، دومينو، طاولة الزهر | رزمة Klondike مستقلة، أحجار فريدة، البار والضرب واستخدام النرد |
| مضافة في V187 | جاكارو، ليخة | حركة القطع والبطاقات الخاصة؛ تمرير ثلاث أوراق، ليخة الإلزامية ونقاط الفرق |

## تجربة اللاعب

- ترتيب يدوي للورق عبر السحب والإفلات مع الحفاظ على النسخ المتكررة.
- حوار اختيار ورق المجموعة بدل الاعتماد على اقتراح المحرك فقط.
- تنزيل مجموعة أو عدة مجموعات، والتركيب على مجموعات الفريق وفق نمط اللعبة.
- تلوين مستقل لكل مجموعة منزلّة وإظهار المالك والقيمة.
- صور أصلية جديدة لجاكارو وليخة ضمن الهوية الزمردية–الذهبية.

## الأمان والنزاهة

- معرفات الألعاب المجهولة تفشل مغلقة.
- حركات اللعب وانتهاء الوقت داخل معاملات تقفل صف الغرفة وتتحقق من رقم المراجعة.
- التحقق الخادمي يطبق الحمولة المنظمة كاملة على نسخة حالة قبل اعتمادها.
- أيدي الخصوم وseed التوزيع والحالة الداخلية للمحرك محجوبة عن العميل.
- اللعب المحلي لا يستطيع اعتماد شراء أو اقتصاد أو نتيجة مصنفة.

## الفحوص المنفذة

- نجاح عقد الإصدار، سياسة الجذر النظيف، وعقود V170–V187 الثابتة.
- نجاح اختبار V183 التنفيذي بعد إصلاح صاحب الدور في تركس.
- نجاح محاكاة PHP 8.3 مستقلة لكل المحركات العشرين حتى 260 انتقالًا مع فحص الحركات المعلنة ورفض اللعب خارج الدور.
- نجاح اختبار إجهاد من 400 بداية عشوائية (20 بداية لكل لعبة) واختبار محولات المحركات العشرين.
- نجاح تحليل الصياغة المستقل لكل ملفات PHP البالغ عددها 317 ملفًا وكل ملفات Dart البالغ عددها 32 ملفًا.
- نجاح فحص تعارضات Git وبنية المصدر.

## بوابة النشر

بيئة الإنشاء الحالية لا تحتوي Flutter SDK وLaravel/Composer كاملين. لذلك تبقى `flutter analyze` و`flutter test` و`php artisan test` وبناء Android/iOS/Web في GitHub Actions شروطًا إلزامية قبل النشر. لا يثبت هذا التقرير اختبارات أجهزة حقيقية أو تحميل واسع أو اختبار اختراق خارجي.

## حدود المطابقة

القواعد مبنية بصورة أصلية على القواعد المنشورة للألعاب، وليست نسخًا لكود أو تصميم تطبيق آخر. إعدادات غرف متقدمة قد تختلف باختلاف البلد أو النادي؛ يجب تقديمها كخيارات قواعد مستقلة واختبارها قبل تفعيلها في اللعب المصنف.
