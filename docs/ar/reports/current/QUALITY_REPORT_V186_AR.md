## الأمان والنزاهة

- مفاتيح الألعاب المجهولة تفشل مغلقة.
- التوزيع الافتراضي يستخدم seed عشوائيًا مشفرًا بطول 256 بت مع commitment، ولا تصل قيمة seed للاعب قبل نهاية المباراة.
- بيانات أيدي الخصوم والحالة الداخلية `_global_engine` و`_tarneeb_v2` لا تُرسل في الحالة العامة.
- الحركة المنظمة تُنفذ على نسخة داخل الذاكرة للتحقق من الحمولة كاملة قبل الاعتماد.
- الحركة وانتهاء الوقت في واجهتي الهاتف والويب مقفلان بصف الغرفة ومعاملة، ويضيف مسار الهاتف قيمة `_revision`، لمنع الكتابة المتزامنة الضائعة.
- اختبارات النزاهة تتحقق من رفض اللعب خارج الدور ومن حفظ أعداد الورق/الأحجار.

## بوابة النشر

بيئة العمل الحالية لا تحتوي Flutter SDK ولا Laravel/Composer كاملين. تم تشغيل محركات PHP المستقلة فعليًا عبر PHP 8.3 محمول، لكن يبقى `flutter analyze` و`flutter test` و`php artisan test` داخل GitHub Actions بوابة إلزامية قبل APK/AAB أو النشر.

## حدود المطابقة الصريحة

V186 يثبت اكتمال مسار القواعد الأساسي للمحركات الـ18، وأضاف تدبيل تركس وشراء حكم البلوت المرحلي. لا يدّعي نسخ تطبيق جواكر أو مطابقته برمجيًا. خيارات البلوت المتقدمة مثل أشكل، المشاريع المعلنة، دبل/ثلاثة/أربعة/قهوة، ومفتوح/مغلق تحتاج ملف قواعد غرف مستقل واختبارات إضافية قبل وصفها بأنها مطابقة.
