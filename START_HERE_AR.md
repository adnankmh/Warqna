# ابدأ من هنا — Warqnaa V0.3.6+187

1. انسخ محتويات المشروع إلى مستودع GitHub مع إبقاء مجلد `.git` فقط من النسخة السابقة.
2. لا تضع ملفات ZIP أو ملاحظات شخصية جديدة في جذر المشروع. يقبل الفحص فقط ملفات الترقية التاريخية V142/V143 المعروفة بالاسم.
3. نفّذ Commit ثم Push.
4. افتح تبويب Actions وانتظر نجاح جميع مسارات البناء.
5. استخدم `CHECK_WARQNA_WINDOWS.bat` للفحص المحلي على Windows.

أهم تحديثات هذه الحزمة:
- إصلاح فشل `validate_release.py` عند وجود ملفات حزمة V142/V143 القديمة، مع استمرار رفض أي فوضى غير معروفة.
- إصلاح اختبار V183 الذي كان ينفذ عقد تركس بمالك مملكة غير صاحب الدور.
- 20 لعبة ظاهرة، تعمل محليًا مع بوتات دون اشتراط الخادم، وتعمل تنافسيًا بمحركات Laravel موثوقة.
- إضافة جاكارو وليخة بمحركين مستقلين، وصور أصلية متناسقة مع هوية التطبيق.
- تنزيل عدة مجموعات، تركيب على المجموعات، ترتيب يدوي لورق هاند وبناكل، ولون مختلف لكل مجموعة.
- لوحة إدارة موسعة بتبويب سلامة المحركات وحالة Offline/Online وإصدار القواعد.
- قفل تزامني لحركات الخادم، إخفاء معلومات الخصوم، ورفض معرفات الألعاب غير المعروفة.
